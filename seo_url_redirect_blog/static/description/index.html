Now make your website blog more SEO Friendly by rewriting your website URLs and make your website blog pages more searchable. Odoo SEO-URL Redirect/Rewrite For Blog helps you to redirect Old URL to new URL of the Blog.

User Guide

Now make your website blog more SEO Friendly by rewriting your website URLs and make your website blog pages more searchable. This module also provides redirect URL feature to your website blog. Redirect feature helps to send an informative message to your user's browser which will tell them that the page is moved and it will automatically switch to a new page of your choice.

Features:

  • Feature to redirect from source URL to destination URL on your website.
  • Feature to add multiple redirect list.
  • Feature to rewrite blog URL.
  • Feature to bulk rewrite blog URL.
  • Feature to add suffix for blog URL.
  • Feature to remove "/blog" from blog URL to make it SEO Friendly.
  • Feature to automatically redirect for blog products.
  • Feature to rewrite blog post URL.
  • Feature to bulk rewrite blog post URL.
  • Feature to add suffix for blog post URL.
  • Feature to remove "/blog/blog-name/post" from blog post URL to make it SEO Friendly.
  • Feature to automatically redirect for blog posts.
Note: This module is an addon for SEO-URL Redirect/Rewrite
SEO-URL Redirect/Rewrite will only work with the Odoo default theme. To make it work with the customized theme or any third party theme you can contact us.

Default Configuration

Admin can set redirect as well rewrite setting

  • Enable suffix setting to add suffix along with URL.
  • User can add suffix for blog/blog posts as well.
  • Enable "use web server rewrite" setting to remove "/blog" / "blog/blog-name/post" from blog URLs.
